# Clinical Research Portfolio Project — eCRF Design Simulation

## Project Overview

This project is a mock Clinical Data Management (CDM) and Electronic Case Report Form (eCRF) design simulation created to demonstrate practical understanding of clinical trial workflows used in modern clinical research environments.

The project simulates a Phase II Type 2 Diabetes Mellitus clinical trial and focuses on how patient data is collected, validated, and managed using structured eCRF modules.

This portfolio project was developed for educational and professional skill development purposes only.
